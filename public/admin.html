Cấu hình Giá Bán & Kiếm Lợi Nhuận

Bạn có thể cài đặt giá bán hiển thị trên giao diện của khách hàng tại đây. Khi khách hàng quét mã VietQR, họ sẽ trả mức giá này. Sau khi thanh toán thành công, hệ thống chỉ khấu trừ **Giá gốc nhà cung cấp (Muacert)** trong ví đại lý của bạn để tạo cert, mang lại lợi nhuận trực tiếp cho bạn!

💡 Quy Trình Hoạt Động & Lợi Nhuận

Mô hình chênh lệch giá giúp bạn thu hồi vốn và phát triển kinh doanh Cert cho iOS cực kỳ hiệu quả mà không cần ôm hạ tầng hay máy chủ Apple Developer.

Ví dụ quy trình thực tế:

  1. Bạn cấu hình giá bán **Gói Cơ Bản là 80,000đ** (lãi 30,000đ so với giá gốc đại lý).
  2. Khách vào trang chủ của bạn đăng ký, quét mã VietQR và chuyển khoản **80,000đ** trực tiếp vào số tài khoản ngân hàng của bạn.
  3. Cổng **Pay2S.vn** nhận được giao dịch, tự động bắn Webhook thông báo về hệ thống.
  4. Hệ thống nhận được webhook 80k, tự động kết nối API Muacert và kích hoạt gói **Basic (khấu trừ đúng 50k số dư gốc của bạn trên Muacert)**.
  5. Giao dịch hoàn tất! Khách nhận được Cert để cài đặt, ví Muacert của bạn bị khấu trừ 50k, và tài khoản ngân hàng của bạn thực tế đã thu về **80,000đ** (mang lại 30k lợi nhuận chênh lệch ngay lập tức).

🔐 Khóa Kết nối & Tài khoản Admin

Thiết lập các khóa API bảo mật lấy từ Muacert.com và Pay2S.vn, cùng tài khoản đăng nhập admin đại lý.

Lưu ý quan trọng

  • Mỗi lần đổi Token hoặc Key, hãy bấm "Lưu toàn bộ cấu hình bảo mật" để ghi vào server.
  • URL IPN tự động cập nhật theo domain hiện tại của bạn.

📥 Xuất dữ liệu (Backup)

Tải xuống toàn bộ cấu hình hệ thống, danh sách gói cước, cài đặt API và tất cả tài khoản người dùng (bao gồm cả lịch sử mua cert, UDID, trạng thái kích hoạt) dưới dạng file JSON.

📤 Nhập dữ liệu (Restore)

Khôi phục hoặc chuyển dữ liệu từ website cũ sang website mới. Chọn file backup JSON đã tải xuống trước đó. Lưu ý: Thao tác này sẽ ghi đè toàn bộ dữ liệu hiện tại!
